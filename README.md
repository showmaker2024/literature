# 文献仓库说明

这个仓库用于整理个人阅读与收集的论文 PDF，并按专题维护中文笔记、配图和阶段性索引。

## 主要内容

- `文献摘要.md`：根目录索引页，用来汇总当前已经建立摘要的专题入口。
- 各专题目录：按研究问题归档论文、图片与对应笔记。
- `协作备忘.md`：记录协作中的高频操作、命名约定与常见问题处理方案。

## 目录导览

- `分叉/`：微血管分叉中的红细胞分配、hematocrit partition、Zweifach-Fung 效应等。
- `微流控/`：狭窄通道、微流体器件、细胞穿越收缩结构等问题。
- `受限几何中的集体运动/`：受限通道中的 train、phonon、有序化与失稳。
- `active matter/`：微藻游动、非互易相互作用与活性物质相关主题。
- `红细胞黏附/`：糖萼、聚集、双细胞黏附与相关形态演化。
- `红细胞+剪切流/`：剪切流中的红细胞形态、黏度异常与悬液流变。
- `红细胞+直管道/`：直管道 Poiseuille 流中的红细胞簇、队列与水动力相互作用。
- `红细胞+颗粒/`：红细胞背景流中的颗粒捕获、边缘化与输运。
- `群体智能与运动/`：自驱动粒子、相变与群体运动基础文献。
- `植物/`：植物相关综述与暂存资料。
- `颗粒-网络/`：网络结构中的颗粒输运、扩散与分布。
- `扩展引文/`：与主线问题相关的综述和延伸阅读。
- `分裂与生长/`、`实验佐证/`：用于补充特定问题的专题归档。

## 维护约定

- 新增文献时，优先放入最贴近主题的目录中。
- 已形成稳定笔记的专题，应在对应目录内维护 `文献摘要.md`，并同步更新根目录 `文献摘要.md` 索引。
- 若某个目录目前只有 PDF 归档、尚未建立摘要，可先保留目录，待开始整理后再补索引入口。
- 若遇到可复用的整理流程、编码问题或提交/同步故障，需要同步更新 `协作备忘.md`。
- 根目录说明只记录稳定结构与维护方式，不写一次性的阅读进度。
