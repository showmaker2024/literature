# 文献仓库说明

这个仓库用于整理个人阅读过的文献、归档 PDF，并持续维护对应的中文笔记与配图。

## 主要内容

- `文献摘要.md`：持续追加的主笔记文档，按阅读进度记录论文要点。
- 各专题目录：按研究方向分类存放论文与补充资料，如 `分叉/`、`血管网络/`、`红细胞黏附/`、`扩展引文/`。
- `协作备忘.md`：记录协作过程中的高频操作、命名约定与常见问题处理方案。

## 目录导览

- `分叉/`：聚焦微血管分叉中的红细胞或颗粒分配、Zweifach-Fung 效应及相关建模文献。
- `扩展引文/`：存放与主线问题相关的延伸阅读，当前包含 `biochemical wave/`、`metachronal wave in ciliary arrays/`、`non-reciprocal oscillator networks/`、`PCP/` 等子主题。
- 其他专题目录：按问题域继续归档，例如细胞黏附、微流控、血管网络与受限几何中的集体运动。

## 维护约定

- 新增文献时，优先放入最贴近主题的目录中。
- 完成阅读后，优先在 `文献摘要.md` 中补充对应条目；若只是阶段性整理，应明确标注为部分笔记。
- 若遇到可复用的整理流程、编码问题或提交/同步故障，需要同步更新 `协作备忘.md`。
- 当某个目录新增了稳定的子主题结构时，应同步更新本文件，避免后续查找成本上升。
